5.1.2  Write a program to calculate the total marks, aggregate percentage, and grade of a student based on marks in four subjects. The grade is determined as follows:
# Read four subject marks (space-separated)
m1, m2, m3, m4 = map(int, input().split())

# Calculate total and aggregate percentage
total = m1 + m2 + m3 + m4
aggregate = total / 4

# Print total
print(total)

# Print aggregate with 2 decimal places
print(f"{aggregate:.2f}")

# Determine and print grade
if aggregate > 75:
    print("Distinction")
elif aggregate >= 60:
    print("First Division")
elif aggregate >= 50:
    print("Second Division")
elif aggregate >= 40:
    print("Third Division")
else:
    print("Fail")
